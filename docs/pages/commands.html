Command Reference

Complete reference for all Linear CLI commands. Every command outputs JSON by default. Add --human for readable tables. Error responses include hints and usage examples to help AI agents self-correct.

Global Flags

--human - Human-readable output instead of JSON
--api-key - Override API key for this command
--team - Override default team for this command

Issue Commands

Create, manage, and search issues in your Linear workspace.

linear issue create create

Create a new issue.

Flags

FlagDescriptionRequired
--title, -tIssue titleYes
--description, -dIssue description (markdown)No
--teamTeam key (e.g., ENG)Yes*
--assignee, -aAssignee username or 'self'No
--priorityPriority (0-4: none, urgent, high, medium, low)No
--estimateEstimate pointsNo
--label, -lLabel name (repeatable)No
--state, -sInitial state nameNo
--projectProject ID or nameNo
--parent, -pParent issue identifierNo
--due-dateDue date (YYYY-MM-DD)No

* Required unless default team is configured

linear issue create --title "Fix login bug" --team ENG --priority 2

linear issue create \
  --title "Add OAuth support" \
  --description "Implement Google and GitHub OAuth" \
  --team ENG \
  --assignee self \
  --label feature \
  --label auth
linear issue list read

List issues with optional filters.

Flags

FlagDescription
--teamFilter by team key
--state, -sFilter by state type (repeatable): triage, backlog, unstarted, started, completed, canceled
--all-statesInclude all states
--assigneeFilter by assignee
--all-assignees, -AInclude all assignees
--unassigned, -UOnly unassigned issues
--limitMaximum results (default: 50)
--sortSort order: manual, priority
 
linear issue list --team ENG --human

linear issue list --team ENG --state started --state unstarted

linear issue list --team ENG --assignee self --limit 10
linear issue view <identifier> read

View issue details.

Flags

FlagDescription
--webOpen in web browser
--appOpen in Linear app
--no-commentsDon't include comments
 
linear issue view ENG-123

linear issue view ENG-123 --human

linear issue view ENG-123 --web
linear issue update <identifier> update

Update an existing issue.

Flags

Same as issue create (all optional, at least one required)

linear issue update ENG-123 --state "In Progress"

linear issue update ENG-123 --priority 1 --assignee john
linear issue delete <identifier> delete

Delete an issue.

linear issue delete ENG-123
linear issue search <query> read

Search issues by text.

Flags

FlagDescription
--teamFilter by team
--limitMaximum results
--include-archivedInclude archived issues
--include-commentsSearch in comments too
 
linear issue search "authentication bug"

linear issue search "login" --team ENG --limit 10 --human
linear issue comment create/read

Manage issue comments.

# Create a comment
linear issue comment create ENG-123 --body "Fixed in commit abc123"

# List comments
linear issue comment list ENG-123 --limit 10
linear issue attachment create/read/delete

Manage issue attachments.

# Create an attachment
linear issue attachment create ENG-123 \
  --title "Design Doc" \
  --url "https://docs.google.com/..."

# List attachments
linear issue attachment list ENG-123

# Delete attachment
linear issue attachment delete ENG-123 ATTACHMENT_ID
linear issue relate create/delete

Manage issue relationships.

# Create relationship
linear issue relate ENG-123 ENG-456 --blocks
linear issue relate ENG-123 ENG-456 --related-to
linear issue relate ENG-123 ENG-456 --duplicate-of

# Remove relationship
linear issue unrelate ENG-123 ENG-456

# List relationships
linear issue relations ENG-123

Project Commands

Create and manage projects with milestones and status updates.

linear project create create

Create a new project.

Flags

FlagDescriptionRequired
--nameProject nameYes
--descriptionProject descriptionNo
--contentProject content (markdown)No
--teamTeam key (repeatable)Yes*
--statusInitial statusNo
--leadProject lead usernameNo
--priorityPriority (0-4)No
--start-dateStart date (YYYY-MM-DD)No
--target-dateTarget date (YYYY-MM-DD)No
--with-docCreate associated documentNo
--doc-titleDocument title (with --with-doc)No
 
linear project create --name "Q1 Auth Overhaul" --team ENG

linear project create \
  --name "API Redesign" \
  --team ENG \
  --lead john \
  --target-date 2025-03-31 \
  --with-doc \
  --doc-title "API Redesign Spec"
linear project list read

List projects.

Flags

FlagDescription
--teamFilter by team
--all-teamsInclude all teams
--statusFilter by status
--limitMaximum results
 
linear project list --human

linear project list --team ENG --status "In Progress"
linear project view <id> read

View project details.

linear project view PROJECT_ID --human
linear project update <id> update

Update a project.

linear project update PROJECT_ID --status "In Progress"

linear project update PROJECT_ID --target-date 2025-04-15
linear project delete <id> delete

Archive a project.

linear project delete PROJECT_ID
linear project restore <id> update

Restore an archived project.

linear project restore PROJECT_ID
linear project milestone create/read/update/delete

Manage project milestones.

# Create milestone
linear project milestone create PROJECT_ID \
  --name "Phase 1: OAuth" \
  --target-date 2025-02-15

# List milestones
linear project milestone list PROJECT_ID

# Update milestone
linear project milestone update MILESTONE_ID --name "Phase 1 Complete"

# Delete milestone
linear project milestone delete MILESTONE_ID
linear project update-status create/read

Manage project status updates (changelog).

# Create status update
linear project update-status create PROJECT_ID \
  --body "Completed OAuth integration" \
  --health onTrack

# List status updates
linear project update-status list PROJECT_ID

Initiative Commands

Strategic planning with initiatives that group related projects.

linear initiative create create

Create a new initiative.

Flags

FlagDescriptionRequired
--name, -nInitiative nameYes
--description, -dDescriptionNo
--content, -cContent (markdown)No
--status, -sStatus: Planned, Active, CompletedNo
--owner, -oOwner usernameNo
--target-date, -tTarget date (YYYY-MM-DD)No
 
linear initiative create --name "Q1 Platform Improvements"

linear initiative create \
  --name "Security Hardening" \
  --status Active \
  --owner john \
  --target-date 2025-03-31
linear initiative list read

List initiatives.

linear initiative list --human

linear initiative list --status Active
linear initiative view <id> read

View initiative details.

linear initiative view INIT_ID --human
linear initiative update <id> update

Update an initiative.

linear initiative update INIT_ID --status Completed
linear initiative archive <id> delete

Archive an initiative.

linear initiative archive INIT_ID
linear initiative restore <id> update

Restore an archived initiative.

linear initiative restore INIT_ID
linear initiative project-add/remove update

Manage initiative projects.

# Add project to initiative
linear initiative project-add INIT_ID PROJECT_ID

# Remove project from initiative
linear initiative project-remove INIT_ID PROJECT_ID

Label Commands

Manage issue labels.

linear label list read

List labels.

linear label list --team ENG --human
linear label create create

Create a new label.

Flags

FlagDescription
--name, -nLabel name (required)
--description, -dDescription
--color, -cColor (#RRGGBB)
--team, -tTeam (for team labels)
--parent, -pParent label ID
--is-groupCreate as label group
 
linear label create --name "bug" --color "#FF0000" --team ENG
linear label update <id> update

Update a label.

linear label update LABEL_ID --name "critical-bug" --color "#FF0000"
linear label delete <id> delete

Delete a label.

linear label delete LABEL_ID

Auth Commands

Manage authentication.

linear auth login auth

Authenticate with Linear.

Flags

FlagDescription

linear auth login --api-key lin_api_xxxxx

linear auth login --api-key lin_api_xxxxx --team ENG
linear auth logout auth

Remove stored credentials.

linear auth logout
linear auth status auth

Check authentication status.

linear auth status
linear whoami auth

Display current user info.

linear whoami

linear whoami --human

Config Commands

Manage CLI configuration.

linear config setup setup

Interactive setup wizard.

Flags

FlagDescription
--api-keyAPI key
--teamDefault team
--stdinRead API key from stdin
--validateValidate key without storing
 
# Interactive
linear config setup

# Non-interactive (for AI agents)
echo "lin_api_xxxxx" | linear config setup --stdin --team ENG
linear config set <key> <value> config

Set a config value.

linear config set team_key ENG
linear config get <key> config

Get a config value.

linear config get team_key
linear config list config

List all config values.

linear config list
