Changelog

All notable changes to the Linear CLI. This project follows Semantic Versioning.

v0.1.0

2025-12-25 Initial Release

Added

Initial release of the Linear CLI - a JSON-first command-line interface for Linear, designed for AI agents.

Core Commands

  • Issue - Create, list, view, update, delete, and search issues
  • Project - Manage projects with milestones and status updates
  • Document - Create, view, update, delete, and search documents
  • Initiative - Strategic planning with initiatives that group projects

Utility Commands

  • Team - List teams and team members
  • User - List and search users
  • Label - Create, list, update, and delete labels
  • Workflow - List and cache workflow states

Auth & Config

  • auth login/logout/status - Secure keychain-based authentication
  • config setup - Interactive setup wizard with stdin support
  • config set/get/list - Manage configuration
  • whoami - Display current user info

Features

  • JSON-first output - Structured output for AI agents
  • --human flag - Readable tables for terminal use
  • Error hints - Helpful hints and usage examples in error responses
  • 24-hour caching - Workflows, statuses, users, and labels
  • Secure credential storage - System keychain integration
  • Stdin support - Non-interactive setup for AI agents
